# origo-bc-mcp

Origo Business Central MCP server — install package.

> **This repo is automatically updated by the Azure Pipelines build.**
> Do not edit manually — changes will be overwritten on next build.

## Install

```bash
npm install -g github:businesscentralal/origo-bc-mcp
```

## Setup

```bash
origo-bc-mcp-server setup
```

This walks you through configuring your BC connection and VS Code `mcp.json`.

## Update

```bash
npm install -g github:businesscentralal/origo-bc-mcp
```

## Source

The source code lives in Azure DevOps (Origo-BC-PTE / BC-PTE-CloudEvents / Cloud Events MCP).
