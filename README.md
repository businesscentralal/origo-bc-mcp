# origo-bc-mcp

Origo Business Central MCP server — install package.

## Install

```bash
npm install -g github:businesscentralal/origo-bc-mcp
```

## Setup

```bash
origo-bc-mcp-server setup
```

## Update

```bash
npm install -g github:businesscentralal/origo-bc-mcp
```

See [setup docs](docs/setup-from-tarball.md) for detailed instructions.
